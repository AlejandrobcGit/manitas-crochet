# Requisitos Funcionales

Proyecto: 🧶 Manitas Crochet

Subtítulo:
> Creaciones únicas tejidas con imaginación

---

# RF-01 Visualizar catálogo

El sistema debe permitir visualizar todas las figuras registradas.

## Criterios de aceptación

- Mostrar imagen principal.
- Mostrar nombre.
- Mostrar categoría.
- Mostrar dificultad.

---

# RF-02 Ver detalle de figura

El usuario podrá consultar la información completa de una figura.

## Criterios de aceptación

- Nombre.
- Descripción.
- Categoría.
- Dificultad.
- Materiales.
- Autor.

---

# RF-03 Buscar figuras

El usuario podrá buscar figuras por nombre.

---

# RF-04 Filtrar por categoría

Categorías disponibles:

- Animales
- Anime
- Videojuegos
- Fantasía
- Películas
- Navidad
- Halloween
- Originales

---

# RF-05 Filtrar por dificultad

Niveles:

- Principiante
- Intermedio
- Avanzado

---

# RF-06 Crear figura

Permitir registrar nuevas figuras.

---

# RF-07 Editar figura

Permitir modificar figuras existentes.

---

# RF-08 Eliminar figura

Permitir eliminar figuras del catálogo.

---

# RF-09 Gestionar materiales

Cada figura podrá contener una lista de materiales.
